# Paved Road: AI Enablement for PE Portfolio Companies

This repository is the working toolkit for an AI-enablement practice serving mid-market
private equity firms and their portfolio companies ($10-50M revenue B2B SaaS and
tech-enabled services). It pairs an operating playbook — a 30/60/90-day forward-deployed
engineering model and a pre-acquisition AI diligence scorecard — with a runnable reference
implementation: a semantic interoperability layer over messy enterprise data, a
cost-governed multi-agent orchestration engine, and an evaluation harness that answers
"how do we know it's right" with numbers instead of assurances.

Everything here runs on a laptop against synthetic-but-realistic enterprise data, works
with any LLM provider behind a thin adapter, and treats cost discipline as a design
constraint rather than a feature. The playbook documents are written for deal teams and
operating partners; the code is written for portfolio-company CTOs who want to see the
patterns before they trust them. This README will be rewritten with measured results once
the reference implementation is complete.
